# 🎬 My Pocket Movie

> 내가 찜한 영화, 나만의 컬렉션으로 정리하는 영화 커뮤니티 플랫폼

---

## 📌 프로젝트 소개

**My Pocket Movie**는 사용자가 TMDB API를 통해 원하는 영화를 검색하고,  
리뷰 작성 및 평점 등록, 좋아요, 댓글, 위시리스트 등 다양한 소셜 기능을 통해  
나만의 영화 컬렉션을 만들어가는 커뮤니티 기반 플랫폼입니다.

---

## 🚀 주요 기능

- 🎥 TMDB API 기반 영화 검색 및 상세 조회  
- ✍️ 영화 리뷰 및 평점 등록  
- 💬 댓글, 좋아요, 콘텐츠 공유 등 소셜 기능  
- 📌 관심 영화 북마크 및 컬렉션 관리  
- 🔐 JWT 기반 사용자 인증/인가  

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| **Backend** | Java 17, Spring Boot, Spring Security, JPA, JWT, OAuth2 (Google/Kakao) |
| **Infra** | AWS EC2, RDS(MySQL), Docker, ECR, GitHub Actions, CodeDeploy |
| **API 문서화** | Swagger, Postman |
| **기타** | RestClient, Parameter Store, DTO 패턴, 계층형 아키텍처 적용 |

---

## 📁 프로젝트 아키텍처

- 계층형 구조 (Controller → Service → Repository → Entity/DTO)
- RESTful API 기반 설계
- Swagger를 활용한 API 문서 자동화
- GitHub Actions + CodeDeploy를 활용한 CI/CD 자동화
- Parameter Store를 이용한 환경 변수 관리

## 🌱 우리 팀의 성장 목표

> “이제 이론은 충분하다. 개발자다운 서비스를 만들고 싶다!”

- 기술 구현을 넘어 **문제 해결 중심의 개발자**로 성장  
- **기획 → 설계 → 구현 → 배포** 전 과정을 직접 경험  
- 각자의 강점과 개성을 살린 협업을 통해 완성도 높은 결과물 도출
