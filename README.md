# 🎬 My Pocket Movie (PM 포트폴리오)

> **“영화 애호가들을 위한 전용 커뮤니티 MVP”**  
> 6주간 **문제 정의 → 기획 → 설계 → 실행 → 피드백 개선** 전 과정을 PM으로 주도.

---

## 📌 프로젝트 개요
- **문제 정의**
  - 범용 SNS에는 영화 특화 커뮤니티 부재 → 팬들은 흩어진 채널에서 소통
  - 기존 리뷰 플랫폼은 단편적 기록 중심 → 개인화·소셜 상호작용 부족
  - → 영화 애호가의 **취향 기반 기록·공유**가 가능한 전용 서비스 필요
- **프로젝트 목적**
  1) **개인화**: 선호 영화·리뷰 중심 컬렉션 구축  
  2) **소셜**: 댓글·좋아요로 상호작용 강화  
  3) **데이터 기반 UX**: 검색→상세→리뷰 플로우 개선  
  4) **서비스 목표**: 재방문율 20% 이상을 위한 피드백 기반 고도화
- **기간/팀**: 2025.05~07(6주) / PM 1, BE 3, FE 1

---

## 🚀 역할 및 주요 기여 (PM Role)
- **기획·요구사항 정의**
  - 사용자 여정 정의, 핵심 기능(PRD) 문서화
  - **RICE** 점수화로 우선순위·MVP 범위 확정
  - **Acceptance Criteria** 수립 → 스코프 크립 0건
- **일정·리스크 관리**
  - 주 단위 **Sprint 로드맵** 설계·리뷰 진행
  - API 제한·병목 리스크 사전 식별, **캐싱 전략** 적용 → 장애 0건
- **피드백 기반 개선**
  - 사용자 20명 테스트: “검색 정확도 부족, 리뷰 작성 복잡” 도출
  - 개선: 리뷰 단계 3→2로 축소, 검색 자동완성 적용
  - 재검증: **리뷰 완료율 +9%p, 검색 성공률 +12%p**
- **협업 프로세스**
  - **Notion(SSOT)**: 회의록·PRD 24h 내 기록 → 히스토리 누락 0건
  - **GitHub Issues/Kanban**: 이슈 단위 관리 → 진행률 100% 가시화
  - **Slack**: 실시간 이슈 공유·결정사항 Notion 이관 → **PR 리드타임 30% 단축**

## 📊 프로젝트 성과
| 항목 | 개선 전 | 개선 후 | 결과 |
|---|---:|---:|---:|
| 리뷰 작성 완료율 | 41% | 50% | **+9%p** |
| 검색 성공률 | 68% | 80% | **+12%p** |
| PR 리뷰 리드타임 | 기준 | 기준-30% | **30% 단축** |
| 협업 품질(중요 PR 오류) | 4건 | 0건 | **100% 개선** |

> *성과는 사용자 테스트·협업 지표 중심. 단순 타임라인(“6주 배포”)은 성과에서 제외.*
